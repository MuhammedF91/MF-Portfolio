## About This Portfolio

This repository is a curated summary of my hands‑on DevOps experience, combined with an overview of my professional background in application support. It intentionally does **not** include source code for any of the projects — the focus is on the tools, workflows, and real‑world problems I've tackled.

I built this portfolio to:

- Document the DevOps tools and practices I've adopted day‑to‑day.
- Showcase real implementations without exposing private source code.
- Track my growth and learning path in the DevOps field.
- Give recruiters and technical teams a clear, fast read on what I can do.
